PRODUCTION USE
==============

For real employee ID cards:
- Use actual employee photos
- Recommended resolution: 1000 x 1000 pixels or higher
- Any background works (auto-removed)
- Name files by Employee ID (not employee name)
- Follow same ZIP structure


QUESTIONS?
==========

The HR ID Card Generator app provides detailed error messages
and validation feedback. Follow the error messages to fix any issues.

Most common issue: Filename doesn't match Employee ID exactly
Solution: Ensure filenames match IDs in CSV (case-sensitive)
